Klonirati git repozitorij EdunovaPP24 na lokalni disk putm Github Desktop programa
u datoteci u ovom projektu vhost.txt prilagoditi putanje na svom računalu
u datoteku C:\xampp\apache\conf\extra\httpd-vhosts.conf dodati sadržaj datoteke vhost.txt
u datoteku C:\Windows\System32\drivers\etc dodati liniju 127.0.0.1   edunovaapp.xyz
napraviti restart apache servera


U ovom repozitoriju će biti kod s nastave.
